Обработка ошибок

Демонстрируется возможность обработать ошибку перед отображением её пользователю.

В модуле приложения реализован обработчик отображения ошибки. В обработчике для ошибки прав добавляется информация, о том, к кому обратиться при возникновении такой ошибки.

Для примера, следует зайти под пользователем "Продавец" и попробовать открыть обработку редактирование цен. Появится окно об ошибке, в котором будут указаны контакты специалиста, который поможет решить проблему.

Дополнительно есть возможность декларативной настройки текстов сообщений об ошибках. Настроить тексты можно с помощью стандартной обработки "Управление настройками обработки ошибок" (открывается через функции для технического специалиста). В данной базе уже настроен шаблон текста для ошибок хранимых данных. Получить такую ошибку можно, например, попробовав записать элемент справочника с неуникальным кодом.
