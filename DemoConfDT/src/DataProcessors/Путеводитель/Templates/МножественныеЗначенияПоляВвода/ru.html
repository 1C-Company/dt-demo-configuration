Множественные значения поля ввода

Использование множественных значений поля ввода демонстрируется в форме элемента справочника Исходящее письмо. 

Введите в поле «Адрес» произвольный адрес электронной почты и нажмите клавишу Enter. Добавится множественное значение в виде отдельного элемента интерфейса, при этом позиция курсора останется внутри поля ввода. 

Адреса электронной почты можно вводить по одному или через разделитель: точку с запятой.

Также множественные значения используются в форме справочника Товары для ввода множественных видов характеристик (например, цвет).
